# 3D Solar System Simulator

React, Three.js, and React Three Fiber로 만든 한국어 3D 태양계 관측 앱입니다.

## Features

- 태양과 8개 행성의 3D 궤도 시뮬레이션
- 교육용으로 압축한 거리/크기 표현
- 재생, 일시정지, 리셋, 시간 속도 조절
- 행성 선택 시 카메라 포커스 이동
- 실제 거리, 지름, 공전 주기, 자전 주기 정보 패널
- 데스크톱과 모바일 반응형 UI

## Getting Started

```bash
npm install
npm run dev
```
